# DevOps E-commerce Source Code

This is a simple e-commerce source-code-only application for DevOps training.

## Developer handoff

The developer only provides source code. No Dockerfile, no Kubernetes manifests, no Helm chart, and no CI/CD pipeline are included.

## Run locally

```bash
npm install
npm start
```

Open:

```text
http://localhost:3000
```

## Health check

```bash
curl http://localhost:3000/health
```
